🔎 개발기간 : 2021년 10월 30일 ~ 2021년 12월 11일(예정)

**Core Value**

> 🙌 공유하다 - 에코슈머라면 누구든지 아이코를 통해, 가치있는 소비 스타일링을 공유할 수 있습니다. <br>

>👍 가치있다 - 에코 스타일샷을 통해 유저는 가치있는 라이프스타일을 기록하고, 친환경 제품에 대한 정보를 얻을 수 있습니다.<br>
